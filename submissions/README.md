# Submissions

This folder contains Kaggle-ready prediction outputs for women, men, and a combined submission.

## Women
- Folder: `submissions/women/2026/`
- Key files:
  - `submission.csv`: Kaggle submission (all 2026 pairwise matchups)
  - `WNCAATourneyPredictions_with_preds.csv`: same predictions, explicit probabilities
  - `WNCAATourneyPredictions.csv`: winner/loser pairs derived from probabilities
  - `bracket_2026_results.csv`, `bracket_2026_visual.md`, `bracket_2026_tree.txt`
  - `championship_total.csv`

## Men
- Folder: `submissions/men/2026/`
- Key files:
  - `submission.csv`: Kaggle submission (all 2026 pairwise matchups)
  - `MNCAATourneyPredictions_with_preds.csv`: same predictions, explicit probabilities
  - `bracket_2026_results.csv`, `bracket_2026_visual.md`, `bracket_2026_tree.txt`
  - `championship_total.csv`

## Combined (Men + Women)
- File: `submissions/submission.csv`
- This is the combined Kaggle submission containing both men and women matchups.
